Jogo do Dinossauro no Chrome

Projeto desenvolvido durante um bootcamp feito na DIO onde recriei sob orientação do professor o jogo do dinossauro que o Chrome disponibiliza quando estamos sem internet. Nesse projeto foi usado HTML, CSS e JavaScript e abordado diversos conceitos introdutórios importantes para programação na web como tags básicas HTML, manipulação de eventos, funções, manipulação de elementos HTML usando JavaScript, estilização e animações básicas com CSS.
